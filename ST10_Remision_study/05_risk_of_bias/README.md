# 05_risk_of_bias — RoB 2 assessment

- **`rob2_template.csv`** — one row per study × outcome; five domain judgements + overall.
- **`rob2_guidance.md`** — domain questions, the overall-judgement algorithm, and
  review-specific notes (medication-cessation verification, cluster extension).

Two assessors independently rate `low` / `some_concerns` / `high` per domain; reconcile.
The completed CSV feeds `../06_analysis/R/09_rob_grade_viz.R` (robvis plots).
