# Project timeline

Anchored to PROSPERO registration (29 May 2026). The binding constraint is the
protocol's commitment to run the **final search within 30 days of registration**
(≈ 28 June 2026).

| Phase | Target window | Owner | Gate to clear before next phase |
|-------|---------------|-------|----------------------------------|
| 0. Infrastructure + team onboarding | Jun 2026 | NWA | Repo set up (done); team agrees roles; librarian + supervisor confirmed |
| 1. Finalise + PRESS-review search | by ~20 Jun 2026 | NWA, Lib | PRESS checklist signed off |
| 2. Run all 13 database searches + grey lit | by **28 Jun 2026** | NAO, NWA | Search log complete; exports archived |
| 3. Deduplicate + pilot screening | late Jun 2026 | NWA, JO | **Cohen's κ ≥ 0.75** |
| 4. Title/abstract + full-text screening | Jul 2026 | NWA, JO | PRISMA counts finalised; excluded-with-reasons logged |
| 5. Data extraction (duplicate) + author contact | Jul–Aug 2026 | NWA, NAO | Extraction reconciled; `meta_input.csv` built |
| 6. Risk of bias (RoB 2) | Aug 2026 | NWA, JO | RoB CSV complete |
| 7. Meta-analysis + sensitivity/subgroups | Aug 2026 | NWA, NAO | Results locked in `06_analysis/results/` |
| 8. GRADE + SoF tables | Aug–Sep 2026 | NWA, JO | Certainty agreed per outcome |
| 9. Manuscript draft | Sep 2026 | NWA (all review) | Co-author sign-off |
| 10. Submission (+ medRxiv preprint, OSF deposit) | Oct 2026 | NWA | Data/code on OSF; ORCIDs collected |

> Dates are planning targets, not commitments — the only hard deadline is the 30-day
> final-search window from registration. Update this table at each team check-in.
