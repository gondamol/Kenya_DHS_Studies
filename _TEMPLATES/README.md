# Templates

This folder contains shared manuscript-formatting assets used by the study workflow.

Included files:

- `reference_doc.docx`: Word reference style for Quarto manuscript rendering
- `suppress-docx-authors.lua`: Quarto filter used during manuscript rendering
- `vancouver.csl`: Vancouver citation style
- `manuscript_template.qmd`: base Quarto manuscript template
- `make_reference_doc.R` and `make_reference_doc.py`: helper scripts for reference document generation
- `manuscript-html-head.html`: optional HTML head fragment for Quarto outputs
